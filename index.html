Befehle

Befehl Beschreibung
~ Userseite
pwd Zeigt aktuellen Arbeitsorder an
ls Zeigt die Inhalte des aktuellen Ordners in Spalten an
ls -1 Zeigt die Inhalte des aktuellen Orders in Listenform an
ls -a Zeigt auch versteckte Datein an
ls -l Zeigt Zusatzinformationen an
clear Löscht den Bildschirm
cd+Order Springt in den angegebenen Order: zum Beispiel cd Dokumente. Auch mehrere Order mit / möglich. Vereinfachung: mit TAB sucht Terminal die möglichen Dateien/ Ordner aus
cd .. Springt eine Orderebene zurück
mkdir Make Directory - Neuer Ordner! Auch mehrere Order parallel: mkdir assets/css
sudo Irgendwas mit Passworteingabe
touch Datei erstellen
mv Verschieben. Befehl: mv style.css /assets/css/
cp Kopieren. cp index.html /assets/
exit Speichern, Prozess beendet
rm Remove/ Löschen von Datein: rm index.html
rmdir Löschen von Ordnern
&& Befehle kombinieren und hintereinander ablaufen lassen
nano Texteingabe über Nano
sh Shell ausführen, z.B. sh ordnerstruktur.sh
hier noch mehr...

Shortcuts

  • Pfeiltasten Innerhalb eines Orders bewegen
  • CMD + U Löscht die Linie vom Cursorpunkt zurück zum Anfang
  • CMD + A Bewegt den Cursor an den Anfang der Zeile
  • CMD + E Bewegt den Cursor ans Ende der Zeile
